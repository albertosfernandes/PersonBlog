Este primeiro post começo falando sobre a infraestrutura DNS. Decidi utilizar a AWS solução para resolução de nomes pois todo o projeto está sobre a nuvem AWS e terei muitos ganhos com os recursos avançados do Route53. Além também de manter todo o gerenciamento centralizado.

Base da infraestrutura

Onde quero chegar na construção dessa infraestrutura.

Construindo meu nome na internet

Primeiro passo é definir um nome que representa o projeto e pesquisar sobre a disponibilidade, para isso existem diversas ferramentas que podem te auxiliar, meu caso resolvi utilizar o registrobr.

Basta acessar o site "https://registro.br/", a primeira opção que deve aparecer é "PESQUISAR DOMÍNIO", após pesquisar a disponibilidade do nome escolhido deve definir a categoria:

  • Atividades comerciais: com.br
  • Tecnologia: tec.br
  • Blogs: blog.com

Estes são alguns exemplos, existem diversos, abuse da criatividade!

Com esses tópicos definidos, é seguir o passo a passo para finalizar o seu registro.

Importante lembrar ao leitor que para ter o registro válido na internet existe uma taxa que precisa ser paga.

AWS Route53

Com o nome definido o próximo passo é criar um servidor de nomes "DNS".

Este servidor vai guardar todas as configurações e apontamentos para o nome registrado. Ele vai dizer onde está o web-site, o servidor de e-mail, ou qualquer outro recurso do projeto.
